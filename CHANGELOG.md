### Added

- Initial template setup
- Vue Router with lazy-loaded routes
- Pinia state management (theme + locale store)
- vue-i18n with English and Turkish locales
- Tailwind CSS v4 with custom theme
- Typed HTTP client and useApi composable
- Theme composable (dark/light/system)
- AppButton reusable component
- Vitest + @vue/test-utils testing setup
- ESLint 9 flat config with vue plugin
- Prettier with Tailwind plugin
- Husky + lint-staged pre-commit hooks
- Auto-imports for Vue, Router, Pinia, and vue-i18n APIs
- Auto-registration for components in src/components/
